2. PROJECT OVERVIEW

Design a responsive desktop website (1440 px) for Desa Petung, a small village located on the foothills of Mount Merapi, Indonesia.

This is NOT a government portal.

This is a branding website designed to introduce the village, tell its story, showcase its people, and promote Gumuk Petung Camp as the village's main tourism destination.

The website should feel like a professionally crafted destination-branding website created by a design studio, not an AI-generated landing page.

The experience should prioritize storytelling, authenticity, whitespace, photography, and emotional engagement.

3. WEBSITE GOALS

The interface should successfully achieve these goals:

• Introduce Desa Petung to first-time visitors.

• Build trust and emotional connection.

• Showcase the beauty of the village.

• Promote Gumuk Petung Camp.

• Encourage visitors to explore more pages.

• Create a memorable first impression.

• Encourage camping reservations through WhatsApp.

The website should never feel like a government information portal.

Instead, it should feel welcoming, inspiring, calm, and authentic.
