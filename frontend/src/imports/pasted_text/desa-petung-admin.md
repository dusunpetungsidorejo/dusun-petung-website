Act as a Senior Product Designer, UX Designer, and Design System Specialist specializing in modern content management systems and administrative dashboards.



Design a premium desktop Admin Dashboard (1440px) for the Desa Petung website.



This dashboard is intended for village administrators to manage website content. It should prioritize usability, clarity, and efficiency while maintaining the exact same visual identity as the public Desa Petung website.



The dashboard should feel like it was designed by the same design team, using the same design system, colors, typography, spacing, and components.



---



## Theme Consistency



The public Desa Petung website has already been designed.



Use the attached website design as the PRIMARY visual reference.



Maintain consistency with the existing website by using the same:



- Color palette

- Typography

- Spacing system

- Grid system

- Border radius

- Shadows

- Button styles

- Input styles

- Card styles

- Icon style

- Design language

- Overall visual hierarchy



The transition between the public website and this dashboard should feel seamless.



Someone who has seen the public website should immediately recognize that this dashboard belongs to the same product ecosystem.



Do NOT copy the exact page layouts from the public website.



Instead, reinterpret the same visual language into an efficient administrative interface.



---



## Dashboard Purpose



This dashboard is designed for village administrators to manage the content displayed on the Desa Petung website.



The interface should be simple enough for users with basic computer skills.



Navigation should be intuitive, forms should be straightforward, and every action should feel easy to understand.



---

Create a left sidebar containing:



- Dashboard

Create a top navigation bar containing:



- Search

---



## Visual Direction



Use the exact same visual personality as the public website.



The dashboard should feel:



- Warm

- Natural

- Friendly

- Modern

- Clean

- Professional



Use:



- generous whitespace

- rounded corners (12–16px)

- subtle shadows

- clean typography

- simple iconography

- balanced layouts



Avoid making everything look like cards.



Use visual rhythm similar to the public website.



---



## UX Principles



Prioritize:



- Easy scanning

- Clear hierarchy

- Large click targets

- Simple navigation

- Consistent spacing

- Accessibility

- Efficient content management



Forms should be straightforward and require minimal learning.



---



## Responsive Considerations



Focus on desktop (1440px).



Ensure the layout can later be adapted for tablet without major redesign.



---



## Important Constraints



Do NOT design a generic admin dashboard.



Do NOT use:



- Bootstrap Admin

- Material Dashboard

- Generic SaaS Dashboard

- Corporate ERP styling



Avoid:



- Glassmorphism

- Neumorphism

- Heavy gradients

- Oversized icons

- Bright saturated colors

- Decorative shapes without purpose

- Excessive animations

- Cluttered layouts



The dashboard should preserve the calm, natural, and welcoming identity of Desa Petung while remaining practical for daily administration.

---



## Final Quality Checklist



Before completing the design, review it carefully.



Ask yourself:



- Does this clearly belong to the same ecosystem as the Desa Petung website?

- Is the visual identity consistent?

- Is the interface intuitive?

- Is the hierarchy clear?

- Would village administrators easily understand how to use it?

- Does it feel handcrafted rather than AI-generated?



If the answer to any question is "No", revise the design before considering it complete.
